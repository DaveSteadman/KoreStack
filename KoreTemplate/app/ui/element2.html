Element2 — Component Reference

Two foundational constants: --row-h (topbars, panel headers, toolbar rows) and --el-h (buttons, inputs, all interactive controls). Shape is a theme variable (--e2-radius: 2px), not per-component. This page is the Phase 0 test and reference document — build here, then migrate live pages.

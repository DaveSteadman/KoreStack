Here's a POST /api/save endpoint. It validates the path stays inside the workspace root to prevent path traversal.

python · server.py 
@app.post("/api/save")
async def save_file(req: SaveRequest):
    cfg = get_config()
    target = (cfg.workspace_root / req.path).resolve()
    if cfg.workspace_root not in target.parents:
        raise HTTPException(403)
    target.write_text(req.content, encoding="utf-8")
    return {"saved": True}
